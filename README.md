# Sportstrama Website

Static pages for www.sportstrama.com

## Pages

- `/` - Home/landing
- `/pricing.html` - Premium plans
- `/privacy.html` - Privacy policy
- `/terms.html` - Terms of service
- `/contact.html` - Contact/support
- `/success.html` - Stripe checkout success
- `/cancel.html` - Stripe checkout canceled

## Deploy

Hosted on GitHub Pages. Any push to main triggers automatic deployment.

## DNS Setup

Point your domain to GitHub Pages:
```
www CNAME <your-username>.github.io.
```
